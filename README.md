# spring-boot
Atividade de Spring boot
 
## Atividade: Layout com Bootstrap

Este repositório contém uma página `index.html` que demonstra o uso do Bootstrap (grid, colunas, cards e formulário) conforme a atividade prática.

### Como abrir

- Basta abrir o arquivo `index.html` no seu navegador (clique duplo no Windows ou abra com o navegador de sua preferência).
- Alternativamente, você pode usar uma extensão como "Live Server" no VS Code para visualizar com recarregamento automático.

### O que foi implementado

1. Parágrafo introdutório dentro de um container centralizado.
2. Texto dividido em duas colunas no grid (empilha em telas pequenas).
3. Três cards com imagem e texto (lado a lado em telas grandes, responsivos em telas menores).
4. Formulário de contato com Nome, E-mail, Mensagem e botão de Enviar.

Os elementos utilizam as classes padrão do Bootstrap e estão organizados em `container`, `row` e `col-*` conforme apropriado.
